Tools and Technologies

Techonology Stack

A-Frame

A-Frame is an open-source web framework for building virtual reality experiences. It is maintained by developers from Supermedium and Google. A-Frame is an entity component system framework for Three.js where developers can create 3D and WebVR scenes using HTML.

Animaze

Easily transform yourself at the click of a button and connect with your family, friends, followers, and fans on any platform that works with a webcam. Customize and add props to create your own unique persona. Animaze is the character creator for your online self!

MailChimp

For sending regular updates related to the college we have used mailchimp. Parents will get the information through mails. We have used mailchimp for that purpose. Mailchimp Assistant can help with finding information quickly in Mailchimp so that we can get back to work.

Google Map API

A Google Maps API is an API that allows developers to access Google Maps data and functionality for their own projects.
You can customize Google Maps on your website or application.

HTML/CSS/JS

HTML provides the basic structure of websites. CSS is used for formatting and layout.
JavaScript is used to control the behavior of different elements.

Text To Speech API

The Text-to-Speech API enables developers to generate human-like speech. The API converts text into audio formats such as WAV, MP3, or Ogg Opus. It also supports Speech Synthesis Markup Language (SSML).

Problem Statement

Geotagging of all educational institutions integrated with virtual tour on Bharat Map so that any students/parents can visit any part of the country on map and take virtual tour along with facilities/courses available across the country.

  • Virtual Tour

Our Solution

We provide an integrated solution for both parents and their wards to know about the colleges where they are planning to send their child for the higher educations. We provide the details about the placement scenerios,infrastructure,faculty details so that the parents can have clear picture about the college. we also provide peer comparision of different colleges in India.

  • Walkthrough facility of every college and its infrastructure
  • Insights of libraries, center of excellence and other college facilities
  • Save time and expenses of travelling
  • Simple interface and easy to navigate
  • Solution to Covid crisis
  • Details about staffs and courses
